# Tablet UI 가이드

## 1. 디자인 시스템

### 색상
| 용도 | 색상 | 코드 |
|------|------|------|
| 주요 | 파란색 | #3B82F6 |
| 성공 | 초록색 | #10B981 |
| 경고 | 주황색 | #F59E0B |
| 오류 | 빨간색 | #EF4444 |
| 배경 | 흰색 | #FFFFFF |

### 타이포그래피
- **제목**: 24px, Bold
- **본문**: 16px, Regular
- **캡션**: 12px, Regular

### 간격
- Small: 8px
- Medium: 16px
- Large: 24px

---

## 2. 레이아웃

### 전체 구조
```
┌─────────────────────────────┐
│      Header (64px)          │
├─────────────────────────────┤
│                             │
│      Main Content           │
│                             │
├─────────────────────────────┤
│   Navigation Bar (56px)     │
└─────────────────────────────┘
```

### Safe Area
- Top: 44px (상단 노치 고려)
- Bottom: 34px (하단 홈 바 고려)

---

## 3. 컴포넌트

### 버튼
- **Primary**: 배경색 파란색, 텍스트 흰색
- **Secondary**: 배경색 회색, 텍스트 검정색
- **Danger**: 배경색 빨간색, 텍스트 흰색

### 카드
- 배경색: 흰색
- 테두리: 1px 회색
- 모서리: 8px 둥글음
- 패딩: 16px

### Drawer
- 배경: 흰색
- 너비: 100% (우측에서 슬라이드)
- 상단 여백: 16px

---

## 4. 화면별 가이드

### 대시보드
- KPI 카드 4개 (2x2 그리드)
- 작업 탭 3개 (오늘/금주/긴급)
- 좌측: 작업 목록 (ScrollArea)
- 우측: Google Maps

### 작업 지시
- 상단: 필터 (상태, 유형, 기간)
- 중앙: 작업 목록 (카드 형식)
- 우측: 지도 (선택 가능)

### 정류장
- 상단: 검색 바
- 중앙: 정류장 목록
- Drawer: 상세 정보 (탭: 기본정보, 단말, 점검)

---

## 5. 반응형 설계

### 화면 크기
- 소형 태블릿: 600px 이상
- 중형 태블릿: 768px 이상
- 대형 태블릿: 1024px 이상

### 적응형 레이아웃
- 600px 이상: 좌우 분할 레이아웃
- 600px 미만: 단일 열 레이아웃

---

## 6. 상호작용

### 제스처
- **탭**: 단일 클릭 (작업 선택)
- **길게 누르기**: 컨텍스트 메뉴 표시 (미구현)
- **스와이프**: Drawer 닫기, 목록 새로고침
- **핀치**: 지도 확대/축소

### 애니메이션
- Drawer 슬라이드: 300ms
- 페이드: 200ms
- 없음: 로딩 시간이 짧으면 애니메이션 불필요

---

## 7. 접근성

### 텍스트
- 최소 폰트: 12px
- 명도 대비: 4.5:1 (WCAG AA)

### 터치 대상
- 최소 크기: 44x44px

### 포커스
- 모든 인터랙티브 요소는 명확한 포커스 표시

---

## 8. 상태 표시

### 작업 상태
- **예정**: 회색 배지
- **진행중**: 파란색 배지
- **완료**: 초록색 배지

### 단말 상태
- **정상**: 초록색 마커
- **경고**: 주황색 마커
- **오류**: 빨간색 마커
- **통신중단**: 회색 마커
