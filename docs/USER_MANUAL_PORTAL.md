## 로그인

### 로그인 페이지
- URL: `https://your-domain.com/login`
- 이메일과 비밀번호로 로그인합니다.
- "비밀번호 잊음" 링크로 비밀번호 재설정이 가능합니다.

### 로그인 후
- 할당된 권한에 따라 메뉴가 표시됩니다.
- 우측 상단의 프로필 아이콘으로 계정 설정 및 로그아웃이 가능합니다.

## Registry 모듈

### 개요
Registry는 마스터 데이터(자산, 단말, 고객사, 정류장)를 관리하는 모듈입니다.

### 1. 자산(Assets) 관리

#### 자산 목록 조회
- 경로: `Registry > Assets`
- 필터: 유형, 상태, 고객사, 구간별 조회 가능
- 검색: 자산명, 자산ID로 검색 가능

#### 자산 상세 조회
- 자산 카드 클릭 시 상세 정보 Drawer 오픈
- 내용:
  - 기본 정보 (명칭, ID, 상태)
  - 설치 정보 (설치일, 설치 단말)
  - 비용 정보 (구매가, 설치비)
  - 첨부 파일 (증명서, 사진)

#### 자산 추가/수정
- "추가" 버튼 클릭 또는 자산 선택 후 수정 가능
- 필수 입력: 자산명, 유형, 고객사
- 선택 입력: 설명, 비용, 첨부 파일

---

### 2. 단말(Devices) 관리

#### 단말 목록 조회
- 경로: `Registry > Devices`
- 필터: 상태, 위치, 고객사별 조회
- 실시간 상태 표시 (정상/오류/통신중단)

#### 단말 상세 정보
- 단말 카드 클릭 시 상세 Drawer 오픈
- 내용:
  - 기본 정보 (모델, 일련번호, 상태)
  - 설치 정보 (설치 정류장, 설치일)
  - 이력 (점검, 교체, 버전 업그레이드)
  - 성능 지표 (배터리, 통신 품질)

---

### 3. 고객사(Customers) 관리

#### 고객사 목록
- 경로: `Registry > Customers`
- 필터: 산업, 지역, 계약 상태
- 검색: 회사명, 사업자번호로 검색

#### 고객사 상세 정보
- 담당자 정보, 계약 조건, 관련 자산 목록 조회
- 수정 권한이 있는 경우 정보 업데이트 가능

---

### 4. 정류장(Bus Stops) 관리

#### 정류장 목록
- 경로: `Registry > Bus Stops`
- 지도 또는 테이블 뷰 선택 가능
- 필터: 지역, 설치 상태, 담당 업체

#### 정류장 상세 정보
- 위치 정보, 설치된 단말, 점검 이력
- 점검 기록 추가 가능 (현장팀만 가능)

## Field-Ops 모듈

### 개요
Field-Ops는 설치/유지보수 작업을 관리하고 진행 상황을 추적합니다.

### 1. 작업지시서(Work Orders) 관리

#### 작업지시서 목록
- 경로: `Field-Ops > Work Orders`
- 필터: 상태(계획/진행중/완료), 유형(설치/유지보수/긴급)
- 검색: 고객사, 정류장, 작업 번호

#### 작업지시서 생성
- "새 작업" 버튼 클릭
- 기본 정보 입력:
  - 작업 유형 (설치/유지보수/긴급)
  - 대상 정류장/고객사
  - 배정 업체/기사
  - 예정 날짜/시간
  - 작업 내용 설명

#### 작업 진행 상황 추적
- 상태 변경: 계획 → 진행중 → 완료
- 현장 사진 업로드
- 완료 보고서 작성

---

### 2. 설치(Installations) 관리

#### 설치 프로젝트 목록
- 경로: `Field-Ops > Installations`
- 단계별 조회: 사전점검 → 설치 → 테스트 → 완료

#### 설치 일정 관리
- 날짜별 설치 일정 조회
- 리소스(기사, 장비) 배정 확인

---

### 3. 분석(Analytics) 및 리포트

#### 대시보드
- 경로: `Field-Ops > Dashboard`
- KPI: 금월 설치 수, 유지보수 만족도, 작업 완료율
- 차트: 월간 추이, 유형별 분포

#### 리포트 생성
- 기간별, 유형별, 업체별 리포트 생성
- 내보내기: Excel, PDF 형식

## RMS 모듈

### 개요
RMS(Realtime Management System)는 단말의 실시간 모니터링 및 원격 제어 시스템입니다.

### 1. 모니터링(Monitoring)

#### 대시보드
- 경로: `RMS > Dashboard`
- 실시간 상태 요약:
  - 정상 단말 수
  - 경고 단말 수
  - 오류 단말 수
  - 통신 중단 단말 수

#### 단말 모니터링
- 실시간 위치, 배터리 상태, 화면 표시 상태 조회
- 단말 클릭 → 상세 정보 및 제어 옵션 표시

---

### 2. 장애 관리(Fault Management)

#### 장애 목록
- 경로: `RMS > Faults`
- 우선순위별 정렬 (긴급/높음/중간/낮음)
- 상태별 필터 (미해결/처리중/해결)

#### 장애 대응
1. 장애 목록에서 항목 선택
2. 상세 정보 확인 (시간, 위치, 메시지)
3. 조치 선택:
   - 자동 재시작
   - 원격 업데이트
   - 현장 출동 요청

#### 장애 이력 조회
- 단말별, 기간별 장애 이력 조회
- 해결 시간 및 해결 방법 기록

---

### 3. 원격 제어(Remote Control)

#### 명령어 전송
- 경로: `RMS > Remote Control`
- 대상: 단말 또는 그룹 선택
- 명령어:
  - 화면 켜기/끄기
  - 콘텐츠 갱신
  - 소프트웨어 업데이트
  - 재시작

#### 명령어 예약
- 특정 시간에 명령어 실행되도록 예약
- 반복 설정 가능 (일/주/월)

## CMS 모듈

### 개요
CMS(Content Management System)는 정류장 단말에 표시될 콘텐츠를 관리합니다.

### 1. 콘텐츠 관리

#### 템플릿 관리
- 경로: `CMS > Templates`
- 레이아웃 유형: 공지사항, 광고, 안내
- 템플릿 편집 가능

#### 콘텐츠 생성
- 템플릿 선택 → 텍스트/이미지 입력 → 미리보기 → 게시

#### 콘텐츠 스케줄
- 게시 일시, 종료 일시 설정
- 특정 단말 또는 그룹에만 표시 가능

---

### 2. 정책(Policies) 관리

#### 정책 템플릿
- 경로: `CMS > Policies`
- 표시 우선순위, 화면 비율, 전환 시간 설정

#### 정책 적용
- 정류장 또는 단말 선택 → 정책 적용
- 즉시 또는 예약 적용 가능

---

## Admin 모듈

### 개요
Admin 모듈은 시스템 관리자만 접근 가능한 설정 영역입니다.

### 1. 사용자 관리

#### 사용자 목록
- 경로: `Admin > Users`
- 활성/비활성 사용자 필터 가능

#### 사용자 권한 설정
- 역할 선택: 관리자, 마스터, 운영자, 기술자, 조회자
- 권한 범위 설정 (회사/지역별)

---

### 2. 역할(Roles) 관리

#### 역할 목록
- 경로: `Admin > Roles`
- 기본 역할: 관리자, 운영자, 기술자, 조회자

#### 역할 권한 커스터마이징
- 각 기능별 권한 설정
- 데이터 접근 범위 설정

---

### 3. 감사 로그

#### 감사 로그 조회
- 경로: `Admin > Audit Logs`
- 사용자별, 기간별 로그 조회
- 조회자: 언제, 누가, 무엇을 했는지 확인

## 자주 묻는 질문

**Q1: 비밀번호를 잊었습니다.**
- 로그인 페이지에서 "비밀번호 잊음" 클릭
- 등록된 이메일로 재설정 링크 받음
- 새 비밀번호 설정 가능

**Q2: 작업지시서를 수정하려면?**
- Work Orders 목록에서 수정할 작업 선택
- 상태가 "계획" 상태일 때만 수정 가능
- "진행중" 이상은 관리자만 수정 가능

**Q3: 단말이 응답하지 않습니다.**
- RMS > Faults에서 해당 단말의 장애 확인
- "자동 재시작" 명령 시도
- 장애 지속 시 현장 출동 요청

**Q4: 콘텐츠를 특정 단말에만 표시하려면?**
- CMS > Content에서 콘텐츠 생성
- "대상 선택"에서 특정 정류장 선택
- 게시
