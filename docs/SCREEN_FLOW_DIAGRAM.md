## 2. Portal 화면 흐름

### 2.1 메인 네비게이션 구조

```
Portal 메인 레이아웃
│
├── Registry (자산 등록 관리)
│   ├── /registry/dashboard ─────────────────── 대시보드
│   ├── /registry/assets ────────────────────── 자산 관리
│   │   ├── /registry/assets/warehouses ────── 창고 관리
│   │   ├── /registry/assets/inout ─────────── 입출고 관리
│   │   └── /registry/assets/lifecycle ──────── 수명주기 관리
│   ├── /registry/devices ───────────────────── 단말 관리
│   ├── /registry/customers ─────────────────── 고객사 관리
│   ├── /registry/partners ──────────────────── 협력사 관리
│   └── /registry/stops ─────────────────────── 정류장 관리
│
├── Field-Ops (현장 운영)
│   ├── /field-operations/dashboard ─────────── 대시보드
│   ├── /field-operations/work-orders ───────── 작업지시서
│   ├── /field-operations/installations ─────── 설치 관리
│   ├── /field-operations/reports ───────────── 보고서
│   └── /field-operations/analytics ─────────── 분석
│
├── RMS (실시간 모니터링)
│   ├── /rms/dashboard ──────────────────────── 대시보드
│   ├── /rms/devices ────────────────────────── 단말 모니터링
│   │   └── /rms/devices/[deviceId] ─────────── 단말 상세
│   ├── /rms/incidents ──────────────────────── 장애 관리
│   ├── /rms/alerts ─────────────────────────── 알림 관리
│   ├── /rms/commands ───────────────────────── 원격 명령
│   └── /rms/map ────────────────────────────── 지도 뷰
│
├── CMS (콘텐츠 관리)
│   ├── /cms/dashboard ──────────────────────── 대시보드
│   ├── /cms/templates ──────────────────────── 템플릿 관리
│   ├── /cms/contents ───────────────────────── 콘텐츠 관리
│   ├── /cms/deployment ─────────────────────── 배포 관리
│   ├── /cms/approval ───────────────────────── 승인 관리
│   └── /cms/editor/[id] ────────────────────── 에디터
│
├── Analysis (분석)
│   ├── /analysis/prediction ────────────────── 예측 분석
│   ├── /analysis/anomaly ───────────────────── 이상 탐지
│   └── /analysis/lifecycle ─────────────────── 수명 분석
│
└── Admin (관리자)
    ├── /admin/dashboard ────────────────────── 대시보드
    ├── /admin/users ────────────────────────── 사용자 관리
    ├── /admin/roles ────────────────────────── 역할 관리
    ├── /admin/scopes ───────────────────────── 범위 관리
    ├── /admin/audit ────────────────────────── 감사 로그
    └── /admin/settings ─────────────────────── 시스템 설정
```

### 2.2 Registry 상세 흐름

```
Registry 모듈
│
├── Assets (자산)
│   │
│   ├── 목록 조회 (/registry/assets)
│   │   ├── [검색/필터] → 목록 갱신
│   │   ├── [행 클릭] → Drawer 상세 보기
│   │   ├── [입고 버튼] → 입고 Dialog
│   │   └── [출고 버튼] → 출고 Dialog
│   │
│   ├── 창고 관리 (/registry/assets/warehouses)
│   │   ├── [창고 선택] → 재고 목록 표시
│   │   ├── [행 클릭] → Drawer 상세 보기
│   │   └── [이동 버튼] → 창고 이동 Dialog
│   │
│   └── 입출고 (/registry/assets/inout)
│       ├── [Tab: 입고/출고] → 탭 전환
│       ├── [신규 등록] → 등록 Dialog
│       └── [이력 조회] → 필터링된 목록
│
└── Devices (단말)
    │
    ├── 목록 조회 (/registry/devices)
    │   ├── [검색/필터] → 목록 갱신
    │   ├── [행 클릭] → Drawer 상세 보기
    │   └── [등록 버튼] → 등록 Dialog
    │
    └── 상세 (Drawer)
        ├── 기본 정보 탭
        ├── 연결 자산 탭
        ├── 유지보수 이력 탭
        └── [수정] → 수정 Dialog
```

### 2.3 RMS 상세 흐름

```
RMS 모듈
│
├── 대시보드 (/rms/dashboard)
│   ├── KPI 카드 → 클릭 시 상세 페이지 이동
│   ├── 장애 현황 차트 → 클릭 시 장애 목록
│   └── 최근 알림 → 클릭 시 알림 상세
│
├── 단말 모니터링 (/rms/devices)
│   ├── 목록 조회
│   │   ├── [상태 필터] → 목록 갱신
│   │   ├── [행 클릭] → 단말 상세 페이지
│   │   └── [원격 명령] → 명령 Dialog
│   │
│   └── 단말 상세 (/rms/devices/[deviceId])
│       ├── 실시간 상태 패널
│       ├── 통신 이력 차트
│       ├── 장애 이력 탭
│       ├── 원격 명령 탭
│       └── [뒤로가기] → 목록
│
└── 장애 관리 (/rms/incidents)
    ├── 장애 목록
    │   ├── [상태 필터] → 목록 갱신
    │   ├── [행 클릭] → Drawer 상세
    │   └── [처리] → 처리 Dialog
    │
    └── Drawer 상세
        ├── 장애 정보
        ├── 처리 이력
        └── [작업지시 생성] → Field-Ops 연계
```

## 3. Tablet 화면 흐름

### 3.1 메인 네비게이션

```
Tablet 메인 레이아웃
│
├── 대시보드 (/tablet)
│   ├── KPI 카드 (4개) ─────────────── 클릭 → 탭 필터 활성화
│   ├── 탭 필터 (오늘/금주/긴급) ────── 작업 목록 필터링
│   ├── 작업 카드 목록 ────────────── 클릭 → Drawer 오픈 + 지도 마커 선택
│   ├── 지도 (Google Maps) ─────────── 마커 클릭 → Drawer 오픈
│   └── Drawer (작업 상세)
│       ├── [작업 지시 이동] ────────── /tablet/install
│       └── [길찾기] ─────────────── 앱 선택 Dialog
│
├── 작업 지시 (/tablet/install)
│   ├── 작업 유형 탭 (설치/유지보수/긴급)
│   ├── 상태 필터 (전체/배정/진행/완료)
│   ├── 작업 카드 목록
│   │   └── [카드 클릭] → Drawer 오픈
│   ├── 지도 뷰 탭 ─────────────────── 작업 위치 마커
│   ├── 캘린더 뷰 탭 ───────────────── 일정별 작업
│   └── Drawer (작업 상세)
│       ├── [작업 시작] → 상태 변경
│       ├── [완료 처리] → 완료 Dialog (사진 첨부)
│       └── [길찾기] → 앱 선택 Dialog
│
├── 정류장 (/tablet/stops)
│   ├── 검색/필터 ─────────────────── 정류장 목록 갱신
│   ├── 정류장 테이블
│   │   └── [행 클릭] → Drawer 오픈
│   └── Drawer (정류장 상세)
│       ├── 기본 정보
│       ├── 설치 단말 정보
│       ├── 점검 일정
│       ├── 점검 이력 타임라인
│       ├── 담당자 정보
│       ├── [점검 기록] → 점검 등록 Dialog
│       └── [길찾기] → 앱 선택 Dialog
│
├── 단말 현황 (/tablet/terminal)
│   ├── 상태 필터 (전체/정상/주의/장애/오프라인)
│   ├── 단말 카드 목록
│   │   └── [카드 클릭] → 단말 상세 페이지
│   └── 단말 상세 (/tablet/terminal/[terminalId])
│       ├── 기본 정보 탭
│       ├── 통신 이력 탭
│       ├── 장애 이력 탭
│       ├── 사진 갤러리 탭
│       ├── [원격 명령] → 명령 Dialog
│       └── [뒤로가기] → 목록
│
└── 재고 관리 (/tablet/inventory)
    ├── 탭 (재고 현황 / 입출고 이력)
    ├── 창고 선택 드롭다운
    ├── 자산 테이블
    │   └── [행 클릭] → Drawer 상세
    ├── [입고 등록] → 입고 Dialog
    ├── [출고 등록] → 출고 Dialog
    └── Drawer (자산 상세)
        ├── 기본 정보
        ├── 이력 타임라인
        └── [상태 변경] → 상태 Dialog
```

### 3.2 대시보드 상세 흐름

```
대시보드 (/) - v2.0
│

## 4. 크로스 모듈 흐름

### 4.1 Portal → Tablet 데이터 흐름

```
Portal                              Tablet
│                                   │
├── Registry/Assets                 │
│   └── 자산 등록/수정 ────────────────→ 재고 관리 갱신
│                                   │
├── Field-Ops/WorkOrders            │
│   └── 작업지시서 생성 ───────────────→ 대시보드 + 작업 지시 갱신
│                                   │
├── RMS/Devices                     │
│   └── 단말 상태 업데이트 ─────────────→ 단말 현황 갱신
│                                   │
└── Registry/Stops                  │
    └── 정류장 정보 수정 ───────────────→ 정류장 목록 갱신
```

### 4.2 Tablet → Portal 데이터 흐름

```
Tablet                              Portal
│                                   │
├── 작업 지시                        │
│   └── 작업 상태 변경 ────────────────→ Field-Ops 작업지시서 갱신
│   └── 작업 완료 처리 ────────────────→ Field-Ops + Registry 갱신
│                                   │
├── 정류장                           │
│   └── 점검 기록 등록 ────────────────→ Registry/Stops 이력 추가
│                                   │
└── 재고 관리                        │
    └── 입고/출고 처리 ────────────────→ Registry/Assets 갱신
```

RMS                                 Field-Ops               Tablet

├── 장애 감지                        │                       │
│   └── 알림 생성                    │                       │
│                                   │                       │

│                                   │   └── 담당자 배정 ────→ 긴급 작업 알림

```

## 5. Dialog/Drawer 컴포넌트 흐름

### 5.1 공통 Dialog 패턴

```
사용자 액션
     │
     ▼
┌─────────────────┐
│ Dialog 트리거   │ (버튼 클릭, 행 클릭 등)
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ Dialog 오픈     │ (상태: open = true)
│ ┌─────────────┐ │
│ │   Header    │ │ (Title + Description)
│ ├─────────────┤ │
│ │   Content   │ │ (Form, Info, Actions)
│ ├─────────────┤ │
│ │   Footer    │ │ (취소/확인 버튼)
│ └─────────────┘ │
└────────┬────────┘
         │
    ┌────┴────┐
    ▼         ▼
[취소]      [확인]
    │         │
    ▼         ▼
Dialog     API 호출 → 성공/실패
닫기           │
               ▼
           Dialog 닫기
           + 목록 갱신
```

### 5.2 공통 Drawer 패턴

```
테이블/카드 클릭
     │
     ▼
┌─────────────────┐
│ Drawer 오픈     │ (상태: open = true, selected = item)
│ ┌─────────────┐ │
│ │   Header    │ │ (제목 + 배지)
│ ├─────────────┤ │
│ │   Sections  │ │ (정보 그룹별 섹션)
│ ├─────────────┤ │
│ │   Actions   │ │ (버튼 그룹)
│ └─────────────┘ │
└────────┬────────┘
         │
    ┌────┴────────────┐
    ▼                 ▼
[닫기(X)]        [액션 버튼]
    │                 │
    ▼                 ▼
Drawer 닫기     Dialog 오픈 또는
selected 초기화  페이지 이동
```
