## 앱 설치 및 로그인

### 설치
1. 앱 스토어에서 "BIS Tablet" 검색
2. 설치 완료 후 앱 열기
3. 로그인 화면에서 계정 정보 입력

### 로그인
- 이메일과 비밀번호 입력
- "로그인" 버튼 클릭
- 할당된 정류장/단말 자동 로드

### 로그아웃
- 좌측 상단 메뉴 → "설정" → "로그아웃"

## 대시보드

### 개요
앱 시작 시 표시되는 화면으로 오늘/금주 작업 현황을 한눈에 파악합니다.

### KPI 카드 (4개)
```
┌─────────┬─────────┬─────────┬─────────┐
│ 오늘    │ 금주    │ 긴급    │ 창고    │
│ 작업 N건│ 작업 N건│ 출동 N건│ 재고 N건│
└─────────┴─────────┴─────────┴─────────┘
```

각 카드 클릭 → 해당 탭 활성화 및 필터 적용

### 작업 탭 (3개)
- **오늘**: 오늘 예정된 모든 작업
- **금주**: 이번 주 예정된 작업
- **긴급**: 긴급 출동 건

### 작업 목록 (좌측)
각 작업 카드에는:
- 정류장명 및 주소
- 단말 ID
- 작업 유형 (설치/유지보수/긴급)
- 상태 (예정/진행중/완료)
- 예정 시간

**카드 클릭** → 우측 Drawer에 상세 정보 표시

### 지도 (우측)
- 선택된 탭의 작업 위치를 파란 마커로 표시
- 장애 정류장은 빨간 마커로 표시
- **마커 클릭** → 해당 작업 선택 + Drawer 오픈
- **핀치 제스처** → 지도 확대/축소

### 빠른 액션 (Drawer 내)
작업 선택 시 우측 Drawer에서:
1. **작업 지시 이동**: `/tablet/install`로 이동
2. **길찾기**: 카카오맵/네이버/구글 중 선택 후 네비게이션

## 작업 지시

### 기능
설치/유지보수/긴급 작업의 상세 관리 및 진행 상황 업데이트

### 작업 목록 조회
- 경로: 메인 메뉴 > 작업 지시
- 필터:
  - 상태: 예정/진행중/완료
  - 유형: 설치/유지보수/긴급
  - 기간: 오늘/이번주/전체

### 작업 상세 정보
작업 카드 클릭 시:
- 기본 정보 (고객사, 단말 ID, 예정 시간)
- 작업 내용
- 담당 기사 연락처
- 현장 사진 (완료된 작업)

### 작업 상태 업데이트

#### 1. 작업 시작
- 작업 카드 클릭 → "작업 시작" 버튼
- 상태 자동 변경: 예정 → 진행중
- 시작 시간 기록

#### 2. 사진 첨부
- "현장 사진 추가" → 카메라 또는 갤러리 선택
- 최대 10장까지 첨부 가능

#### 3. 작업 완료
- 완료 정보 입력:
  - 완료 시간
  - 점검 결과 (정상/이상)
  - 비고
- "작업 완료" 클릭
- 상태 자동 변경: 진행중 → 완료

### 길찾기
- 작업 상세 → "길찾기" 버튼
- 지도 앱 선택: 카카오맵 / 네이버지도 / 구글맵
- 네비게이션 시작

## 정류장 모니터링

### 기능
정류장별 단말 상태, 점검 기록 관리

### 정류장 목록
- 경로: 메인 메뉴 > 정류장
- 지도 또는 테이블 뷰 선택 가능
- 필터: 지역, 상태, 담당 업체

### 정류장 상세 정보

정류장 카드/마커 클릭 시 Drawer 오픈:

#### 기본 정보 섹션
- 정류장명
- 위치 (주소, 좌표)
- 설치 단말 정보

#### 설치 단말 상세
- 모델명
- 전원 유형 (AC/태양광/하이브리드)
- 배터리 잔량 (%)
- 마지막 통신 시간
- 펌웨어 버전

#### 점검 일정
- 다음 점검 예정일

#### 점검 이력 (타임라인)
최근 점검 기록:
- 점검 날짜
- 점검 유형 (정기/긴급/AS)
- 점검 결과
- 점검자

#### 담당자 정보
- 담당 업체명
- 담당 기사명
- 기사 연락처

### 점검 기록 등록

#### 1. 점검 기록 버튼
Drawer 하단의 "점검 기록" 버튼 클릭

#### 2. 점검 정보 입력
- 점검 유형: 정기점검 / 긴급점검 / 현장AS / 설치 / 교체
- 점검 결과: 정상 / 이상없음 / 수리완료 / 부품교체
- 점검 내용: 자유 입력

#### 3. 저장
"저장" 버튼 클릭 → 기록 자동 업로드

## 단말 현황

### 기능
할당된 단말의 실시간 상태 모니터링

### 단말 목록
- 경로: 메인 메뉴 > 단말 현황
- 상태별 표시:
  - 정상 (초록)
  - 경고 (주황)
  - 오류 (빨강)

### 단말 상세 정보
단말 카드 클릭 시:
- 기본 정보 (모델, ID, 상태)
- 위치 (설치된 정류장)
- 배터리 상태
- 화면 표시 내용
- 성능 지표
- 이력 (점검, 교체, 업그레이드)

### 사진 갤러리
단말별 설치/점검/완료 사진 조회

## 재고 관리

### 기능
창고 자산의 입출고 관리 (본사/창고팀만 접근 가능)

### 재고 현황
- 경로: 메인 메뉴 > 재고 관리
- 창고별 재고 현황 조회
- 자산 유형별 분류

### 입고 등록
1. "입고 등록" 버튼 클릭
2. 자산 정보 입력:
   - 자산명
   - 수량
   - 입고 업체
   - 비고
3. "저장" 클릭

### 출고 등록
1. "출고 등록" 버튼 클릭
2. 자산 정보 입력:
   - 자산명
   - 수량
   - 출고 대상 (정류장/기사)
   - 비고
3. "저장" 클릭

### 재고 이력
자산별 입출고 이력 조회:
- 날짜
- 유형 (입고/출고)
- 수량
- 담당자

## 장애 대응

### 앱이 느립니다
1. 설정에서 앱 캐시 삭제
2. 앱 재시작
3. 네트워크 연결 확인

### 로그인 실패
1. 인터넷 연결 확인
2. 계정 정보 재확인
3. 비밀번호 재설정

### 지도가 안 보입니다
1. GPS 권한 확인
2. 위치 서비스 활성화 확인
3. 구글 맵 설정 확인

## 자주 묻는 질문

**Q1: 작업을 다시 예정 상태로 바꿀 수 있나요?**
- 상태를 되돌릴 수 없습니다. 실수 시 관리자에게 문의하세요.

**Q2: 오프라인에서 사진을 첨부할 수 없나요?**
- 오프라인 모드에서도 사진 첨부 가능합니다. 네트워크 복구 후 자동 업로드됩니다.

**Q3: 길찾기에서 특정 지도 앱이 없습니다.**
- 기기에 해당 지도 앱이 설치되어 있어야 합니다. 앱 스토어에서 설치 후 시도하세요.

**Q4: 점검 기록 등록 후 수정할 수 있나요?**
- 등록 후 수정은 불가능합니다. 새로 등록 후 관리자에게 삭제 요청하세요.
