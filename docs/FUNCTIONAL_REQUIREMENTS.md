# 기능 요구사항 정의서 (Functional Requirements Document)

## 1. 소개

이 문서는 BIS(Bus Information System) Portal & Tablet의 기능 요구사항을 정의합니다.

---

## 2. Portal 기능 요구사항

### 2.1 Registry 모듈

#### FR-REG-001: 자산 관리
- **요구사항**: 자산의 CRUD 작업 및 조회
- **기능**:
  - 자산 목록 조회 (필터, 검색, 정렬)
  - 자산 상세 조회 (Drawer)
  - 자산 생성/수정/삭제
  - 자산 상태 변경 (활성/비활성)
  - 첨부 파일 관리 (증명서, 사진)
- **권한**: 관리자, 운영자

#### FR-REG-002: 단말 관리
- **요구사항**: 단말의 CRUD 작업 및 실시간 모니터링
- **기능**:
  - 단말 목록 조회 (상태별 필터)
  - 단말 상세 조회
  - 단말 생성/수정/삭제
  - 단말 상태 표시 (정상/오류/통신중단)
  - 성능 메트릭 조회
- **권한**: 관리자, 운영자

#### FR-REG-003: 고객사 관리
- **요구사항**: 고객사 정보 관리
- **기능**:
  - 고객사 목록 조회
  - 고객사 상세 조회 (담당자, 계약)
  - 고객사 생성/수정/삭제
  - 계약 조건 관리
- **권한**: 관리자

#### FR-REG-004: 정류장 관리
- **요구사항**: 정류장 마스터 데이터 관리
- **기능**:
  - 정류장 목록 조회 (지도/테이블)
  - 정류장 상세 조회
  - 정류장 생성/수정/삭제
  - 설치된 단말 연계
- **권한**: 관리자, 운영자

---

### 2.2 Field-Ops 모듈

#### FR-FOE-001: 작업지시서 관리
- **요구사항**: 설치/유지보수/긴급 작업 관리
- **기능**:
  - 작업지시서 생성 (유형 선택: 설치/유지보수/긴급)
  - 작업지시서 조회 (상태별 필터)
  - 작업지시서 수정/취소
  - 업체/기사 배정
  - 완료 보고서 작성
- **권한**: 관리자, 운영자
- **Tablet 연동**: Tablet에 자동 동기화

#### FR-FOE-002: 설치 프로젝트 관리
- **요구사항**: 설치 프로젝트 진행 상황 추적
- **기능**:
  - 설치 프로젝트 생성
  - 단계별 상태 관리 (사전점검 → 설치 → 테스트 → 완료)
  - 자산 연계
  - 설치 일정 관리
- **권한**: 관리자, 운영자

#### FR-FOE-003: 분석 및 리포트
- **요구사항**: 작업 성과 분석
- **기능**:
  - 대시보드 (KPI: 설치 수, 만족도, 완료율)
  - 기간별 리포트 생성
  - 유형별 리포트 생성
  - 업체별 리포트 생성
  - 내보내기 (Excel, PDF)
- **권한**: 관리자, 운영자

---

### 2.3 RMS 모듈

#### FR-RMS-001: 실시간 모니터링
- **요구사항**: 모든 단말의 실시간 상태 모니터링
- **기능**:
  - 대시보드 (정상/경고/오류/통신중단 단말 수)
  - 단말별 상태 조회 (위치, 배터리, 화면 표시 상태)
  - 실시간 알림 (임계값 기반)
  - 이력 조회 (24시간 이내 이벤트)
- **권한**: 관리자, 운영자, 조회자

#### FR-RMS-002: 장애 관리
- **요구사항**: 장애 감지, 분류, 해결
- **기능**:
  - 장애 자동 감지
  - 장애 우선순위 분류 (긴급/높음/중간/낮음)
  - 장애 목록 조회 (상태별 필터)
  - 장애 대응 이력 기록
  - 해결 시간 추적
- **권한**: 관리자, 운영자

#### FR-RMS-003: 원격 제어
- **요구사항**: 단말을 원격으로 제어
- **기능**:
  - 명령어 전송 (화면 켜기/끄기, 재시작, 업데이트)
  - 단일/그룹 명령어 실행
  - 명령어 예약 (일시, 반복)
  - 명령어 실행 이력 기록
- **권한**: 관리자, 운영자

---

## 3. Tablet 기능 요구사항

### 3.1 대시보드

#### FR-TAB-001: 작업 요약
- **요구사항**: 오늘/금주 작업 한눈에 파악
- **기능**:
  - KPI 카드 (오늘/금주/긴급/재고 작업 수)
  - 작업 탭 필터 (오늘/금주/긴급)
  - 작업 목록 (좌측, 스크롤 가능)
  - 지도 표시 (우측, 마커 클릭 가능)
  - Drawer 상세 정보 표시

#### FR-TAB-002: 지도 연동
- **요구사항**: Google Maps 기반 작업 위치 표시
- **기능**:
  - 작업 위치 마커 (파란색)
  - 장애 정류장 마커 (빨간색)
  - 마커 클릭 → 작업 선택 + Drawer 오픈
  - 지도 확대/축소
  - 마커 팬 (위치 이동)

#### FR-TAB-003: 빠른 액션
- **요구사항**: Drawer에서 작업 관련 액션 제공
- **기능**:
  - "작업 지시 이동" (→ /tablet/install)
  - "길찾기" Dialog (카카오맵/네이버/구글 선택)

---

### 3.2 작업 지시

#### FR-TAB-004: 작업 조회
- **요구사항**: 할당된 작업 조회 및 상태 필터
- **기능**:
  - 작업 목록 조회 (상태별: 예정/진행중/완료)
  - 작업 유형 필터 (설치/유지보수/긴급)
  - 기간 필터 (오늘/이번주/전체)
  - 지도 표시 (작업 위치)

#### FR-TAB-005: 작업 상태 업데이트
- **요구사항**: 현장에서 작업 진행 상황 실시간 업데이트
- **기능**:
  - 작업 시작 ("작업 시작" 버튼)
  - 현장 사진 첨부 (최대 10장)
  - 작업 완료 (완료 정보 입력)
  - 상태 자동 변경 (예정 → 진행중 → 완료)

#### FR-TAB-006: 길찾기
- **요구사항**: 정류장으로의 네비게이션
- **기능**:
  - 지도 앱 선택 Dialog (카카오맵/네이버/구글)
  - 선택 시 해당 앱에서 네비게이션 시작

---

### 3.3 정류장 모니터링

#### FR-TAB-007: 정류장 조회
- **요구사항**: 담당 정류장의 현황 파악
- **기능**:
  - 정류장 목록 조회 (필터: 지역, 상태, 업체)
  - 정류장 상세 정보 (기본정보, 단말 상세, 점검 일정)
  - 점검 이력 조회 (타임라인)
  - 담당자 정보 표시

#### FR-TAB-008: 점검 기록 등록
- **요구사항**: 정류장 점검 기록 등록
- **기능**:
  - 점검 유형 선택 (정기/긴급/AS/설치/교체)
  - 점검 결과 선택 (정상/이상없음/수리완료/부품교체)
  - 점검 내용 입력
  - 기록 저장 (Portal 자동 동기화)

---

### 3.4 단말 현황

#### FR-TAB-009: 단말 모니터링
- **요구사항**: 할당된 단말의 실시간 상태 모니터링
- **기능**:
  - 단말 목록 조회 (상태별: 정상/경고/오류)
  - 단말 상세 정보 (기본정보, 성능 지표, 이력)
  - 사진 갤러리 조회

---

### 3.5 재고 관리

#### FR-TAB-010: 재고 조회
- **요구사항**: 창고 자산 재고 현황 파악
- **기능**:
  - 창고별 재고 조회
  - 자산 유형별 분류
  - 재고 수량 표시

#### FR-TAB-011: 입출고 등록
- **요구사항**: 현장/창고에서 입출고 기록
- **기능**:
  - 입고 등록 (자산명, 수량, 입고 업체)
  - 출고 등록 (자산명, 수량, 출고 대상)
  - 재고 이력 조회

---

### 3.6 오프라인 기능

#### FR-TAB-012: 오프라인 모드
- **요구사항**: 네트워크 미연결 시 기본 기능 지속 사용
- **기능**:
  - 오프라인 자동 감지
  - 캐시된 데이터 조회 (최근 데이터)
  - 점검 기록 오프라인 등록 (Outbox)
  - 네트워크 복구 후 자동 동기화

## 4. 비기능 요구사항

| 요구사항 | 기준 | 설명 |
|---------|------|------|
| **성능** | < 2초 | 페이지 로딩 시간 |
| **응답성** | < 500ms | API 응답 시간 |
| **가용성** | 99.9% | 월 가용성 |
| **보안** | HTTPS | 모든 통신 암호화 |
| **접근성** | WCAG 2.1 AA | 웹 접근성 준수 |
| **호환성** | iOS 14+ / Android 10+ | Tablet 지원 OS |
